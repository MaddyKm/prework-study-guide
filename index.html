HTML

  • HTML is the foundation for a webpage
  • Head element contains info about the webpage that is unseen by the user
  • Body element contains the body of the webpage that is seen by users
  • HTML tag brackets all the codes
  • Title or heading/ bigger text use tag h1 through h6
  • Unordered list use UL with LI inside for list items
  • Breaks to add a line in space use br
  • To create sections for organization use section
  • Hyper link in with a include link in brackets with the opening then text befor the closing bracket
  • You can add header and footer in the body

CSS

  • Cascading Style Sheets
  • Defines colors, aesthetics, visual layout and organization
  • 3 Types of adding in CSS to html
    • Inline CSS: written with html, uses a style attribute. This is quick but a more permanent fix and makes editing harder
    • Internal Style Sheet: embedded using style element in the head element and contains the rules for the full page
    • External CSS Style Sheet: Most common, keeps rules in a separate file, linked using a link element placed in the HTML head element
  • Sheet is added to assets file
  • Selector: defines the element or attributes to which the rules or declarations will apply
  • Declarations: contain the CSS property and the value of that property
    • e.g. color: blue
    • Color is the property, blue is the value
  • Margin property indicates how much space we want around the outside of an element. When a Margin = 0 we want no extra space
  • Padding property adds space around the content inside an element. Increasing padding makes an element bigger
  • See CSS box model
  • * selector applies the rules to all the elements on the page
  • Use different selectors (section, ul, etc) to honr in on specific pieces
  • Sizes are measured in px
  • add code after meta via link using rel attribute to define the relationship and href to specify the location of the style sheet
  • Class attributes allow us to shaare a css rule to any element we choose
  • we added . card (period denotes class selector) in our stylesheet to create the border/ shadowbox around our sections
  • Next we add class="card" in the section element to apply the css rules

Git

  • Git flow is super important to not make changes to the main code on the pages
  • When editing always first check what branch you're in with 'git status'
  • git checkout -b name: Creates a working branch for your project with whatever you name it
  • Pushing to github 1. save file 2. git add -A 3. git commit -m "notes" 4. git push origin branchname
  • From there it will be pushed into your github and you can get reviews/ merge into the main branch

JavaScript

  • Makes your page interactive
  • Variables- a named container that allows you to store data in your code
  • Declare a variable with 'var' keyword and give it a name after
  • Control Flow is the order the code is executed, javascript is read top to bottom
  • Each line of JavaScript is ended with a semicolon
  • Find the Chrome DevTools in the browser, left click, inspect, console
  • If statements: introduces decision making in our code, only is executed if the statement meets a condition or is true (if, else if, else)
  • Arrray: a single variable used to hold a group of data that is usually related in someway
  • Create an array by naming a variable inside straight brackets (You can store different types of data in an array)
  • Strings are contained in quotes
  • You can work on a single piece of an array, they are automatically given an order in which they are listed starting with 0 and moving up, use name[0]
  • For loops, we use a predictable pattern to perform a task on all items in an array by allowing a single code block to be executed over and over
  • For loops contain:
    • Starting point for the loop(if you want the beginning you declare a variable x and git it a value of 0)
    • Condition: as long as variable x is less than the length of the array the loop will run. Use the length built in array property (length property = # of variables in array).
    • Final Statement allows the array to iterate over each time. x++ is a short cut to add 1 each time
  • Functions define and reuse lines of code
  • Must define function (use function keyword then name of function, below the function is contaned in curly brackets) then call function (functionname ())
  • You can have a randomized selectuib with math function, use math.floor and math.random functions together
